Get Started with the Accommodation API

The Accommodation API allows you to search, book, and manage accommodations effortlessly. Follow this guide to integrate the API into your application.

1. Prerequisites

  • Sign up for an API key from the Developer Portal.
  • Ensure your application can make HTTPS requests.
  • Familiarity with RESTful APIs and JSON format.

2. Authentication

All API requests must include an API key in the request header:

Authorization: Bearer YOUR_API_KEY

3. Making Your First API Call

To fetch available accommodations, make a GET request to the following endpoint:

GET https://api.example.com/accommodations

Example Request:


curl -X GET "https://api.example.com/accommodations" \
     -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json"

4. Response Example


{
    "accommodations": [
        {
            "id": "123",
            "name": "Seaside Hotel",
            "location": "Miami, FL",
            "price": "120.00",
            "currency": "USD"
        }
    ]
}
